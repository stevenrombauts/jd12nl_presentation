Automate!

The Power of Webhooks

  • Git: post-commit

  • GitHub : Service Hooks
  • Assembla: post-commit

Voorbeeld

  1. Find the deploy Deploy Hook URL in DeployHQ ("Settings" » "Servers")
  2. Browse to the Service Hooks page on your Github repo ("Admin" » "Service Hooks")
  3. Put the Deploy Hook URL into Post-Receive URLs
  4. Commit your changes; deployment is now automated!

Reverting changes

Version control in your deployment workflow

  • Find your revision hashes
  • Put them in reverse order into your "revisions to deploy" on DeployHQ
  • Changes will be applied in reverse order, rolling back your code!
