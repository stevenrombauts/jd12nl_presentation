Automate!

The Power of Webhooks

  • Git: post-commit

  • GitHub : Service Hooks
  • Assembla: post-commit

Voorbeeld

  1. Zoek de Deploy Hook URL op in DeployHQ ("Settings" » "Servers")
  2. Ga op GitHub naar je Service Hooks page ("Admin" » "Service Hooks")
  3. Vul de Deploy Hook URL in onder Post-Receive URLs
  4. Commit je wijzigingen; deployment gebeurt nu automatisch!
